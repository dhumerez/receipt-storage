## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Missing requireTenant middleware | Files route was missing requireTenant, causing req.companyId = undefined → 403 on all file requests | advisory | Check all new routes include authenticate + requireTenant + requireRole chain |
| nginx regex before proxy | Static asset regex `~* \.(jpg|...)$` matched /api/*.jpg before the /api/ proxy location | advisory | Always use `^~` prefix on /api/ location so it wins over regex locations |
| Raw img src with auth | `<img src>` and `<a href>` don't send Authorization headers | advisory | Always use AuthenticatedImage component for protected file URLs |
