**Create `frontend/src/api/transactions.ts`:**

This module contains ONLY transaction CRUD functions and file URL helper. Notification/approval functions live in `frontend/src/api/notifications.ts` (created by Plan 04).

Interfaces:
```typescript
export interface TransactionLineItem {
  id: string;
  productId: string | null;
  description: string;
  quantity: string;
  unitPrice: string;
  lineTotal: string;
}

export interface TransactionListItem {
  id: string;
  referenceNumber: string;
  clientName: string;
  totalAmount: string;
  initialPayment: string;
  status: 'draft' | 'pending_approval' | 'active' | 'voided' | 'written_off';
  deliveredAt: string | null;
  submittedBy: string;
  createdAt: string;
}

export interface TransactionDetail extends TransactionListItem {
  clientId: string;
  description: string | null;
  clientNotes: string | null;
  internalNotes: string | null;
  items: TransactionLineItem[];
  documents: DocumentInfo[];
}

export interface DocumentInfo {
  id: string;
  filePath: string;
  originalName: string;
  mimeType: string;
  fileSizeBytes: number;
}

export interface CreateTransactionInput {
  clientId: string;
  description?: string;
  deliveredAt?: string;
  initialPayment: string;
  clientNotes?: string;
  internalNotes?: string;
  items: Array<{
    productId?: string;
    description: string;
    quantity: string;
    unitPrice: string;
  }>;
}
```

Functions:
- `createTransaction(input: CreateTransactionInput, files: File[]): Promise<TransactionDetail>` — Build FormData: `formData.append('data', JSON.stringify(input))`, `files.forEach(f => formData.append('files', f))`. Call `apiClient('/api/v1/transactions', { method: 'POST', body: formData, headers: {} })`. CRITICAL: pass `headers: {}` to override apiClient's default Content-Type so browser sets multipart boundary.
- `getTransactions(params: { search?, clientId?, status?, dateFrom?, dateTo? }): Promise<TransactionListItem[]>` — Build URLSearchParams, GET.
- `getTransaction(id: string): Promise<TransactionDetail>` — GET `/api/v1/transactions/${id}`.
- `getFileUrl(filePath: string): string` — Returns `${BASE_URL}/api/v1/files/${filePath}` where BASE_URL = `import.meta.env.VITE_API_URL ?? ''`.

DO NOT include `approveTransaction`, `rejectTransaction`, `getNotifications`, `getUnreadCount`, `markNotificationRead`, `markAllNotificationsRead`, or `NotificationItem` in this file. Those belong to `frontend/src/api/notifications.ts` (Plan 04).
