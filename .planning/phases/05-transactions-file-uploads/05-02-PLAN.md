**Create `backend/src/middleware/upload.ts`:**

```typescript
import multer from 'multer';

const storage = multer.memoryStorage();

export const uploadMiddleware = multer({
  storage,
  limits: {
    fileSize: 10 * 1024 * 1024, // 10MB per file (FR-06.10)
    files: 5,                    // max 5 files per request (FR-06.10)
  },
  fileFilter(_req, file, cb) {
    // Block SVG at MIME level (FR-06.12)
    if (file.mimetype === 'image/svg+xml' || file.originalname.toLowerCase().endsWith('.svg')) {
      cb(null, false);
      return;
    }
    cb(null, true);
  },
}).array('files', 5);
```

**Create `backend/src/services/upload.service.ts`:**

Export `processFile` function with this signature:
```typescript
export async function processFile(
  buffer: Buffer,
  originalName: string,
  companyId: string,
  entityType: 'transactions' | 'payments',
  entityId: string,
): Promise<{ filePath: string; mimeType: string; sizeBytes: number; originalName: string }>
```

Implementation steps inside processFile:
1. Dynamic import file-type (ESM-only, Pitfall 1): `const { fileTypeFromBuffer } = await import('file-type');`
2. Detect magic bytes: `const detected = await fileTypeFromBuffer(buffer);`
3. Define ALLOWED_TYPES: `['image/jpeg', 'image/png', 'image/webp', 'image/heif', 'application/pdf']`
4. If `!detected || !ALLOWED_TYPES.includes(detected.mime)`, throw Error `'Unsupported file type: ${detected?.mime ?? "unknown"}'`
5. Belt-and-suspenders SVG check: if `detected.mime === 'image/svg+xml'`, throw Error `'SVG uploads are not allowed'`
6. For images (`detected.mime.startsWith('image/')`):
   - Run sharp pipeline: `.rotate()` (EXIF fix, FR-06.5) → `.resize({ width: 1920, withoutEnlargement: true })` (FR-06.6) → `.jpeg({ quality: 85 })` (converts HEIC→JPEG, compresses)
   - Set finalMime = 'image/jpeg'
   - Also generate thumbnail: sharp(buffer).rotate().resize({ width: 200, withoutEnlargement: true }).jpeg({ quality: 75 }).toBuffer() (FR-06.13)
7. For PDFs: finalBuffer = buffer as-is, finalMime = 'application/pdf', no thumbnail
8. Generate UUID filename: `crypto.randomUUID()` (Node built-in, no dep)
9. Set ext: finalMime === 'application/pdf' ? 'pdf' : 'jpg'
10. Build directory: `path.join(UPLOAD_DIR, companyId, entityType, entityId)` where `UPLOAD_DIR = process.env.UPLOAD_DIR || '/var/receipts/uploads'`
11. `await fs.promises.mkdir(dir, { recursive: true })`
12. Write main file: `await fs.promises.writeFile(path.join(dir, `${uuid}.${ext}`), finalBuffer)`
13. Write thumbnail (if image): `await fs.promises.writeFile(path.join(dir, `${uuid}_thumb.${ext}`), thumbBuffer)`
14. Return `{ filePath: path.join(companyId, entityType, entityId, `${uuid}.${ext}`), mimeType: finalMime, sizeBytes: finalBuffer.length, originalName }`

Import crypto, path, fs at the top. Import sharp.

**Create `backend/src/routes/files.ts`:**

Export `filesRouter = Router()`.

Endpoint: `GET /:companyId/:type/:entityId/:filename`

1. Extract params: companyId, type, entityId, filename.
2. Tenant check: if `companyId !== req.companyId`, return 403 `{ error: 'Forbidden' }`.
3. Sanitize filename: ensure it matches `/^[a-f0-9-]+(_thumb)?\.(jpg|pdf)$/` — reject anything else with 400.
4. Verify document exists in DB: query documents table where companyId matches AND filePath contains the entityId and filename. If no row found, return 404.
5. Construct absolute path: `path.join(UPLOAD_DIR, companyId, type, entityId, filename)`.
6. Check file exists on disk: `fs.existsSync(fullPath)` — if not, return 404.
7. Set headers: `Content-Disposition: attachment; filename="${filename}"`, `X-Content-Type-Options: nosniff`, `Content-Type` based on extension.
8. Stream file: `res.sendFile(fullPath)`.

**Mount in app.ts:**
Add `import { filesRouter } from './routes/files.js';`
Add `app.use('/api/v1/files', authenticate, filesRouter);` — authenticate only (no requireTenant since companyId is in URL path and verified in handler, client role also needs file access).

Place before the 404 handler.
