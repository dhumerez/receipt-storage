```typescript
export async function processLogo(
  buffer: Buffer,
  companyId: string,
): Promise<{ filePath: string; sizeBytes: number }> {
  const { fileTypeFromBuffer } = await import('file-type');
  const detected = await fileTypeFromBuffer(buffer);
  if (!detected || !['image/jpeg', 'image/png', 'image/webp'].includes(detected.mime)) {
    throw new Error(`Unsupported logo type: ${detected?.mime ?? 'unknown'}`);
  }
  const finalBuffer = await sharp(buffer)
    .rotate()
    .resize({ width: 300, withoutEnlargement: true })
    .jpeg({ quality: 90 })
    .toBuffer();
  const dir = path.join(UPLOAD_DIR, companyId);
  await fs.promises.mkdir(dir, { recursive: true });
  const filePath = path.join(dir, 'logo.jpg');
  await fs.promises.writeFile(filePath, finalBuffer);
  return { filePath, sizeBytes: finalBuffer.length };
}
```

Also add a `deleteLogo` function:
```typescript
export async function deleteLogo(companyId: string): Promise<void> {
  const filePath = path.join(UPLOAD_DIR, companyId, 'logo.jpg');
  await fs.promises.unlink(filePath).catch(() => {}); // ignore if not exists
}
```

**4. Create report.service.ts** with three exported functions:

`getCompanyReport(companyId: string, dateFrom?: string, dateTo?: string, showSettled?: boolean)`:
- Query: Join `clients` LEFT JOIN `debtBalances` on clientId+companyId
- WHERE: `clients.companyId = companyId`, `clients.isActive = true`
- Date filter: If dateFrom/dateTo provided, filter on `debts.createdAt` using a subquery or by joining debts table directly. Use `>= dateFrom` and `< dateTo + 1 day` (exclusive upper bound) to handle timestamp boundary (Pitfall 4 from RESEARCH.md).
- GROUP BY clientId, clientName
- Return array of `{ clientId, clientName, totalDebts: string, totalPaid: string, outstandingBalance: string }`
- If showSettled is false (default), filter out rows where outstandingBalance = '0.00' using HAVING clause
- Use SQL-level SUM/COALESCE for all money math (Pitfall 2 — NUMERIC returns strings)

`getClientReport(companyId: string, clientId: string, dateFrom?: string, dateTo?: string)`:
- Query transactions for clientId+companyId within date range (filter on `transactions.deliveredAt`)
- For each transaction: include its debt (if any) from debtBalances view
- For each debt: include payments array (amount, paidAt, paymentMethod, reference, status)
- Return: `{ client: { id, fullName, email, phone }, transactions: Array<{ id, referenceNumber, deliveredAt, description, totalAmount, items: [...], debt?: { totalAmount, amountPaid, remainingBalance, status, payments: [...] } }> }`

`getReceiptData(companyId: string, transactionId: string)`:
- Query single transaction with line items, client info, debt+payments
- Return: `{ company: { name, logoPath }, client: { fullName, email, phone }, transaction: { referenceNumber, deliveredAt, description, items: [...], totalAmount }, debt?: { totalAmount, amountPaid, remainingBalance, status, payments: [...] } }`

Import from: `drizzle-orm` (eq, and, sql, desc, gte, lt), `../db/client.js` (db), `../db/schema.js` (clients, transactions, transactionItems, debts, debtBalances, payments, companies, products).
