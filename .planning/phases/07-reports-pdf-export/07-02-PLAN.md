**2. Create backend/src/services/pdf/pdf-base.ts** with these exports:

**`streamPdf(res: Response, filename: string, buildFn: (doc: PDFKit.PDFDocument) => void | Promise<void>): Promise<void>`**
- Create `new PDFDocument({ bufferPages: true, size: 'LETTER', margin: 50 })` (per D-12: Helvetica built-in default font)
- Call `await buildFn(doc)` to build all content
- Call `addPageNumbers(doc)` to add footers on all pages
- Set response headers: `Content-Type: application/pdf`, `Content-Disposition: attachment; filename="${filename}"`
- `doc.pipe(res)` then `doc.end()`

**`addPageNumbers(doc: PDFKit.PDFDocument): void`** (per D-10)
- Get buffered page range: `const range = doc.bufferedPageRange()`
- Loop through pages, for each page `doc.switchToPage(i)`:
  - "Page X of Y" centered at y = `doc.page.height - 40`, Helvetica 8pt, color `#9CA3AF`
  - "Generated: YYYY-MM-DD" right-aligned at same y, Helvetica 8pt, color `#9CA3AF`

**`async addCompanyHeader(doc: PDFKit.PDFDocument, companyName: string, logoPath: string | null, reportTitle: string, dateRange?: string): Promise<number>`** (per D-09, UI-SPEC PDF Header)
- If logoPath exists and file is readable: `doc.image(logoBuffer, 50, 50, { width: 80, height: 50, fit: [80, 50] })` -- read file as Buffer first (Pitfall 3: handle ENOENT gracefully, skip logo if file missing)
- Company name: Helvetica-Bold 16pt, color `#111827`, positioned right of logo (or at x=50 if no logo)
- Report title below: Helvetica 12pt, color `#6B7280`
- Date range below title: Helvetica 10pt, color `#6B7280` (if provided)
- Horizontal rule: 0.5pt stroke `#D1D5DB` with 10pt gap below
- Return the y position after the header for content to start

**`drawTable(doc: PDFKit.PDFDocument, columns: TableColumn[], rows: any[], startY: number): number`** (per D-11)
- Define `TableColumn` interface: `{ header: string; width: number; align?: 'left' | 'center' | 'right'; getValue: (row: any) => string }`
- `pageBottom = doc.page.height - 60` (room for footer)
- Draw header row: Helvetica-Bold 9pt, color `#374151`, background fill `#F3F4F6`, 0.5pt bottom border `#D1D5DB`
- For each data row: check if y + 20 > pageBottom, if so `doc.addPage()`, reset y=50, redraw headers (D-11 repeated headers)
- Body text: Helvetica 9pt, color `#111827`, row height 16pt
- Use `doc.text(value, x, y, { width: col.width, align: col.align || 'left' })` to constrain text (Pitfall 5)
- Numbers right-aligned, text left-aligned
- Return final y position

**4. Add PDF streaming endpoints to backend/src/routes/reports.ts:**

Add three new routes (append to existing file):

`GET /company/pdf`:
- Validate query: `{ dateFrom?: string, dateTo?: string, showSettled?: string }`
- Fetch report data via `getCompanyReport(...)`
- Fetch company record (name + logoPath) via `db.select().from(companies).where(eq(companies.id, req.companyId!))`
- Call `streamPdf(res, 'company-report-YYYY-MM-DD.pdf', async (doc) => { await buildCompanyReportPdf(doc, company.name, company.logoPath, data, dateFrom, dateTo) })`

`GET /client/:clientId/pdf`:
- Validate clientId UUID + query params
- Fetch client report data via `getClientReport(...)`
- Fetch company record
- Call `streamPdf(res, 'client-report-{clientName}-YYYY-MM-DD.pdf', async (doc) => { await buildClientReportPdf(doc, company.name, company.logoPath, data, dateFrom, dateTo) })`

`GET /receipt/:transactionId/pdf`:
- Validate transactionId UUID
- Fetch receipt data via `getReceiptData(...)`
- Call `streamPdf(res, 'receipt-{referenceNumber}.pdf', async (doc) => { await buildReceiptPdf(doc, data) })`

All three set `Content-Type: application/pdf` and `Content-Disposition: attachment` (handled by streamPdf).
