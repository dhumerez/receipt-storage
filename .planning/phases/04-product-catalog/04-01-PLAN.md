    ```typescript
    const CreateProductSchema = z.object({
      name: z.string().min(1).max(255),
      description: z.string().optional(),
      unitPrice: z.string().regex(/^\d+(\.\d{1,2})?$/, 'Must be a valid price (e.g. "12.50")'),
      unit: z.string().max(50).optional(),
    });

    const UpdateProductSchema = z.object({
      name: z.string().min(1).max(255).optional(),
      description: z.string().optional().nullable(),
      unitPrice: z.string().regex(/^\d+(\.\d{1,2})?$/).optional(),
      unit: z.string().max(50).optional().nullable(),
    });
    ```

    GET / handler:
    - `const companyId = req.companyId!;`
    - `const { search, status } = req.query as { search?: string; status?: string };`
    - Build `conditions: any[] = [eq(products.companyId, companyId)]`
    - If search: `conditions.push(ilike(products.name, '%' + search.trim() + '%'))`
    - If status === 'active': `conditions.push(eq(products.isActive, true))`
    - If status === 'inactive': `conditions.push(eq(products.isActive, false))`
    - Select all columns, `.where(and(...conditions))`, `.orderBy(products.name)`

    POST / handler:
    - Parse with CreateProductSchema.safeParse; 400 on failure
    - `const companyId = req.companyId!;`
    - Insert with `{ companyId, name, description: description || null, unitPrice, unit: unit || null }`
    - `.returning()` → `res.status(201).json(created)`

    PATCH /:id handler:
    - Parse with UpdateProductSchema.safeParse; 400 on failure
    - Ownership check: `db.select({ id: products.id }).from(products).where(and(eq(products.id, id), eq(products.companyId, companyId))).limit(1)` → 404 if empty
    - `db.update(products).set({ ...parsed.data, updatedAt: new Date() }).where(and(eq(products.id, id), eq(products.companyId, companyId))).returning()` → `res.json(updated)`

    PATCH /:id/deactivate handler:
    - Ownership check (same pattern) → 404 if empty
    - `db.update(products).set({ isActive: false, updatedAt: new Date() }).where(and(...))` → `res.status(204).send()`

    PATCH /:id/reactivate handler:
    - Ownership check (same pattern) → 404 if empty
    - `db.update(products).set({ isActive: true, updatedAt: new Date() }).where(and(...))` → `res.status(204).send()`

    NOTE: companyId must NEVER come from req.body — always from req.companyId! (NFR-01.1). Add a comment on each handler making this explicit.
